Cara kerja mapping:
• ERP 1 = Nama toko di kolom "Nama Cabang" pada file Penerimaan Penjualan
• ERP 2 = Nama toko di kolom "Cabang" pada file Laporan Transaksi
• GrabFood Store Name = Kolom "Store Name" di file laporan GrabFood
• ShopeeFood Store Name = Kolom "Store Name" di file laporan ShopeeFood
• GoFood Merchant ID = Merchant ID (G...) dari file laporan GoFood
• Field yang kosong (null) berarti toko tersebut tidak ada di sumber data tersebut
• Folder Name harus UPPERCASE dan digunakan sebagai identifier utama
